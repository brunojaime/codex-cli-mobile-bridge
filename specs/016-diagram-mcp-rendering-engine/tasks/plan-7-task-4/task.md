# T036 Render SVG diagrams in Flutter with viewport controls

Spec: 016-diagram-mcp-rendering-engine

Plan: Frontend Diagram Viewer

Status: completed

- [x] T036 Render SVG diagrams in Flutter with viewport controls.

## Acceptance Notes

- The diagram detail view renders self-contained SVG content generated by the MCP engine.
- Users can pan, zoom, reset view, and inspect the diagram on mobile-sized screens without clipping labels.
- Malformed or unsupported SVG content shows a controlled error state.

## Implementation Notes

- Prefer the existing Flutter dependency set if an SVG renderer is already present; otherwise add a focused dependency such as `flutter_svg`.
- Wrap the rendered SVG in `InteractiveViewer` or an equivalent stable viewport.
- Do not execute scripts, remote references, or unsafe SVG features.
